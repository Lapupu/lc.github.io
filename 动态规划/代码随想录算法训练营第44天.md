## 337.打家劫舍 III

树状图的打家劫舍，dp数组和迭代和前两道题不一样，题目的分析思路是按照递归来分析的

对于树状结构，dp是`[不偷这个节点的金钱, 偷节点的金钱]`

### 确定递归函数的参数和返回值

返回值是当前节点的dp，终止条件是`cur==NULL`

### 确定遍历顺序

后续遍历，要先算好节点才能继续向根节点规划

### 确定单层递归的逻辑

```cpp
// 偷cur
int val1 = cur->val + left[0] + right[0];
// 不偷cur
int val2 = max(left[0], left[1]) + max(right[0], right[1]);
```
